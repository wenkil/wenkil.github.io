虚拟机首次安装centos系统常见问题

虚拟机首次安装打开centos 系统后的一些常见问题

1.无法查看ip

（原因：网络配置没打开）

vi /etc/sysconfig/network-scripts/ifcfg-ens33

ONBOOT=no  改为 ONBOOT=yes

然后运行service network restart 重启网络

这个时候使用 ping www.baidu.com 测试网络是否可用

然后输入命令 ip addr show 即可查看：

这个解决办法还有一种是在虚拟机里刚开始安装centos系统时选择打开网络开关：


2.yum 命令报错

首先进入到/etc/yum.repos.d

cd /etc/yum.repos.d

然后运行以下命令
mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

再更换repo源
curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

再运行yum clean all

这时候就可以使用yum install xxx 了
